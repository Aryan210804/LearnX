Description: deque from collections module is efficient for queue operations.

Linked Lists

Definition: A linked list is a linear data structure where each element points to the next.

Code Example:

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last = self.head
        while last.next:
            last = last.next
        last.next = new_node

    def print_list(self):
        current = self.head
        while current:
            print(current.data, end=" ")
            current = current.next

ll = LinkedList()
ll.append(1)
ll.append(2)
ll.print_list()  # Output: 1 2

Description: Nodes contain data and a pointer to next node. Linked lists allow efficient insertions/deletions.

Trees

Definition: Trees are hierarchical data structures with nodes connected as parent and children.

Binary Trees

Definition: Each node has up to two children, called left and right child.

Code Example:

class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

root = Node(1)
root.left = Node(2)
root.right = Node(3)
print(root.left.data)  # Output: 2

Binary Search Trees (BST)

Definition: A binary tree where left subtree nodes are less and right subtree nodes are greater than the root.

Code Example:

class BSTNode:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

def insert(root, key):
    if not root:
        return BSTNode(key)
    if key < root.data:
        root.left = insert(root.left, key)
    else:
        root.right = insert(root.right, key)
    return root

root = None
for key in [8, 3, 10, 1, 6]:
    root = insert(root, key)
print(root.left.right.data)  # Output: 6

AVL Trees

Definition: Self-balancing BSTs that keep height difference ≤ 1 for any node.

Description: Balances the tree automatically using rotations to ensure O(log n) operations.

Linear Search

Definition: Searching for an element by checking each item sequentially.

Code Example:

def linear_search(arr, target):
    for i, val in enumerate(arr):
        if val == target:
            return i
    return -1

print(linear_search([5,3,8,6], 8))  # Output: 2

Binary Search

Definition: Efficient search on sorted array by repeatedly dividing search interval in half.

Code Example:

def binary_search(arr, target):
    low, high = 0, len(arr) -1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

print(binary_search([1,3,5,7,9], 7))  # Output: 3

Bubble Sort

Definition: Repeatedly swapping adjacent elements if they are in wrong order.

Code Example:

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]

arr = [5,2,9,1]
bubble_sort(arr)
print(arr)  # Output: [1, 2, 5, 9]

Selection Sort

Definition: Select the minimum element and swap it with the first unsorted element.

Code Example:

def selection_sort(arr):
    for i in range(len(arr)):
        min_idx = i
        for j in range(i+1, len(arr)):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]

arr = [64, 25, 12, 22]
selection_sort(arr)
print(arr)  # Output: [12, 22, 25, 64]

Insertion Sort

Definition: Builds sorted array one element at a time by inserting elements into their correct position.

Code Example:

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key

arr = [12, 11, 13, 5]
insertion_sort(arr)
print(arr)  # Output: [5, 11, 12, 13]

Quick Sort

Definition: Divide-and-conquer algorithm that picks a pivot and partitions the array.

Code Example:

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3,6,8,10,1,2]
print(quick_sort(arr))  # Output: [1, 2, 3, 6, 8, 10]

Counting Sort

Description:
Counting Sort is an integer sorting algorithm that counts the number of occurrences of each distinct element, then calculates their positions, and finally places them in the sorted output. It works efficiently when the range of input data is not significantly larger than the number of elements.

Time Complexity: O(n + k), where n = number of elements, k = range of input values
Space Complexity: O(k)

Python Code:

def counting_sort(arr):
    max_val = max(arr)
    count = [0] * (max_val + 1)
    
    # Count occurrences
    for num in arr:
        count[num] += 1
    
    # Reconstruct sorted array
    index = 0
    for i, c in enumerate(count):
        for _ in range(c):
            arr[index] = i
            index += 1

arr = [4, 2, 2, 8, 3, 3, 1]
counting_sort(arr)
print(arr)  # Output: [1, 2, 2, 3, 3, 4, 8]
Output: [1, 2, 2, 3, 3, 4, 8]

Radix Sort

Description:
Radix Sort processes integers digit by digit, starting from the least significant digit to the most significant digit, using a stable sorting algorithm (often Counting Sort) as a subroutine to sort the digits.

Time Complexity: O(d * (n + k)) where d = number of digits, k = base (usually 10)
Space Complexity: O(n + k)

Python Code:

def counting_sort_for_radix(arr, exp):
    n = len(arr)
    output = [0] * n
    count = [0] * 10
    
    # Count occurrences of digits
    for i in range(n):
        index = (arr[i] // exp) % 10
        count[index] += 1
    
    # Change count[i] so it contains actual position of this digit
    for i in range(1, 10):
        count[i] += count[i-1]
    
    # Build output array
    for i in range(n-1, -1, -1):
        index = (arr[i] // exp) % 10
        output[count[index] - 1] = arr[i]
        count[index] -= 1
    
    for i in range(n):
        arr[i] = output[i]

def radix_sort(arr):
    max_val = max(arr)
    exp = 1
    while max_val // exp > 0:
        counting_sort_for_radix(arr, exp)
        exp *= 10

arr = [170, 45, 75, 90, 802, 24, 2, 66]
radix_sort(arr)
print(arr)  # Output: [2, 24, 45, 66, 75, 90, 170, 802]
Output: [2, 24, 45, 66, 75, 90, 170, 802]

Merge Sort

Description:
Merge Sort is a divide-and-conquer algorithm that divides the array into halves, recursively sorts each half, and then merges the sorted halves.

Time Complexity: O(n log n)
Space Complexity: O(n)

Python Code:

def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left = arr[:mid]
        right = arr[mid:]
        
        merge_sort(left)
        merge_sort(right)
        
        i = j = k = 0
        
        # Merge left and right arrays
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1
        
        # Check for remaining elements
        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1
        
        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1

arr = [12, 11, 13, 5, 6, 7]
merge_sort(arr)
print(arr)  # Output: [5, 6, 7, 11, 12, 13]
Output: [5, 6, 7, 11, 12, 13]
