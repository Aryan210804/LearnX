Java Introduction

Definition: Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible.

Use Cases: Java is widely used for building enterprise-scale applications, Android apps, web servers, and more.

Java Get Started

To start programming in Java, you need:

  • Java Development Kit (JDK) installed.
  • A text editor or IDE (e.g., IntelliJ, Eclipse, VS Code).
  • Basic knowledge of command-line for compiling and running Java programs.

Example: To compile and run HelloWorld.java:

javac HelloWorld.java
java HelloWorld

Java Syntax

Basic Syntax Rules:

  • Java is case sensitive.
  • Every statement ends with a semicolon ;.
  • Programs are structured into classes and methods.
  • Main method syntax: public static void main(String[] args)

Example:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Output

Use System.out.println() to output text with newline, and System.out.print() for text without newline.

Example:

System.out.println("Welcome to Java!");
System.out.print("This is ");
System.out.print("on one line.");
Welcome to Java! This is on one line.

Java Comments

Single-line comment: Starts with //

Multi-line comment: Enclosed between /* ... */

Example:

// This is a single-line comment
/*
  This is a
  multi-line comment
*/

Java Variables

Definition: Variables are containers for storing data values.

Syntax:

dataType variableName = value;

Example:

int age = 25;
String name = "Aryan";

Java Data Types

Java supports:

  • Primitive types: int, float, double, char, boolean, byte, short, long
  • Non-primitive types: Classes, Arrays, Interfaces, Strings

Example:

int number = 10;
double price = 99.99;
char letter = 'A';
boolean flag = true;

Java Type Casting

Definition: Converting a variable from one type to another.

Types:

  • Implicit (widening) casting: smaller to larger type automatically
  • Explicit (narrowing) casting: larger to smaller type manually

Example:

int i = 100;
double d = i; // Implicit casting

double x = 9.78;
int y = (int) x; // Explicit casting

Java Operators

Java supports:

  • Arithmetic operators: +, -, *, /, %
  • Assignment operators: =, +=, -=, etc.
  • Comparison operators: ==, !=, <, >, etc.
  • Logical operators: &&, ||, !

Example:

int a = 10;
int b = 20;
boolean result = (a < b) && (b > 15); // true

Java Strings

Strings are objects that represent sequences of characters.

Example:

String greeting = "Hello, Java!";
int length = greeting.length();

Java Math

The Math class provides methods for mathematical operations.

Example:

double sqrt = Math.sqrt(16);
double pow = Math.pow(2, 3);

Java Booleans

Boolean variables hold true or false values.

Example:

boolean isJavaFun = true;
boolean isFishTasty = false;

Java If...Else

Used for conditional branching.

Syntax:

if (condition) {
    // code if true
} else {
    // code if false
}

Example:

int age = 20;
if (age >= 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
Adult

Java Switch

Used to select one of many blocks of code to be executed.

Syntax:

switch(expression) {
  case value1:
    // code
    break;
  case value2:
    // code
    break;
  default:
    // code
}

Example:

int day = 3;
switch(day) {
  case 1: System.out.println("Monday"); break;
  case 2: System.out.println("Tuesday"); break;
  case 3: System.out.println("Wednesday"); break;
  default: System.out.println("Other day");
}
Wednesday

Java While Loop

Repeats a block of code while a condition is true.

Syntax:

while (condition) {
    // code to execute
}

Example:

int i = 1;
while(i <= 5) {
    System.out.println(i);
    i++;
}
1 2 3 4 5

Java For Loop

Repeats a block of code a fixed number of times.

Syntax:

for (initialization; condition; increment) {
    // code to execute
}

Example:

for (int i=1; i<=5; i++) {
    System.out.println(i);
}
1 2 3

Java For Loop

Used to repeat a block of code a known number of times.

Syntax:

for (initialization; condition; update) {
    // code to execute
}

Example:

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
1 2 3 4 5

Java Break and Continue

Break: Exits the current loop immediately.

Continue: Skips the current iteration and continues with next iteration.

Break Example:

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        break;
    }
    System.out.println(i);
}
1 2

Continue Example:

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }
    System.out.println(i);
}
1 2 4 5

Java Arrays

An array is a container that holds a fixed number of values of a single type.

Syntax:

dataType[] arrayName = new dataType[size];

Example:

int[] numbers = new int[5];
numbers[0] = 10;
numbers[1] = 20;
System.out.println(numbers[0]);  // Outputs 10

Java Methods

A method is a block of code which only runs when it is called. Methods help reuse code.

Syntax:

modifier returnType methodName(parameters) {
    // method body
}

Example:

public void sayHello() {
    System.out.println("Hello!");
}

Java Method Parameters

Parameters allow methods to receive inputs.

Example:

public void greet(String name) {
    System.out.println("Hello, " + name);
}

Java Method Overloading

Same method name with different parameters to perform different tasks.

Example:

public void display(int a) {
    System.out.println(a);
}

public void display(String s) {
    System.out.println(s);
}

Java Scope

Scope refers to the accessibility of variables and methods in different parts of a program.

Types: Local, Instance, Class (Static) scope.

Java Recursion

A method that calls itself to solve smaller instances of a problem.

Example:

public int factorial(int n) {
    if (n == 0) return 1;
    else return n * factorial(n - 1);
}

Java Classes

A class is a blueprint for creating objects, containing fields and methods.

Syntax:

public class ClassName {
    // fields
    // methods
}

Example:

public class Car {
    String color;
    void drive() {
        System.out.println("Driving");
    }
}

Java Object-Oriented Programming (OOP)

Four main principles:

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

Java Constructors

Special methods used to initialize objects.

Example:

public class Car {
    String model;
    public Car(String model) {
        this.model = model;
    }
}

Java Modifiers

Modifiers define the access level and behavior of classes, methods, and variables.

  • Access Modifiers: public, private, protected, default (package-private)
  • Non-access Modifiers: static, final, abstract, synchronized, volatile

Java Encapsulation

Wrapping data (variables) and code (methods) together and restricting access to some components.

Example:

public class Person {
    private String name;
    
    public void setName(String name) {
        this.name = name;
    }
    public String getName() {
        return name;
    }
}

Java Packages and API

Packages group related classes and interfaces. The Java API is a large set of packages and classes provided by Java.

Example of importing a package:

import java.util.Scanner;

Java Inheritance

Mechanism where one class inherits properties and behaviors from another.

Example:

class Animal {
    void sound() {
        System.out.println("Animal makes sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
}

Java Polymorphism

Ability of an object to take many forms. Two types:

  • Compile-time (method overloading)
  • Runtime (method overriding)

Java Inner Classes

Classes defined within another class to logically group classes and control visibility.

Example:

class Outer {
    class Inner {
        void display() {
            System.out.println("Inner class method");
        }
    }
}

Java Abstraction

Hiding complex implementation details and showing only essential features.

Achieved with abstract classes and interfaces.

Java Interface

An interface defines a contract of methods that implementing classes must override.

Example:

interface Vehicle {
    void move();
}

class Bike implements Vehicle {
    public void move() {
        System.out.println("Bike moves");
    }
}

Java Exception Handling

Mechanism to handle runtime errors.

Keywords:

  • try
  • catch
  • finally
  • throw
  • throws

Example:

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero");
}

Java Multithreading

Allows concurrent execution of two or more threads to maximize CPU utilization.

Example:

class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running");
    }
}

public class Test {
    public static void main(String[] args) {
        MyThread t1 = new MyThread();
        t1.start();
    }
}

Java Applets

Small Java programs that run in a web browser (mostly obsolete now).

Java AWT and Swing

AWT (Abstract Window Toolkit) and Swing are used for building graphical user interfaces.

Java JDBC

Java Database Connectivity (JDBC) API allows Java programs to connect to databases.

Java Network Programming

Allows communication between computers over a network using TCP/IP protocols.

Java File Handling

Java provides classes like File, FileReader, FileWriter, BufferedReader, BufferedWriter to work with files.

Java Generics

Enable types (classes and interfaces) to be parameters when defining classes, methods, and interfaces.

Java Lambda Expressions

Provide a clear and concise way to represent one method interface using an expression.

Java Stream API

Used to process collections of objects in a functional style.

Java Records

New feature to create immutable data carriers easily.

Java Modules

Introduced in Java 9, modules group packages and resources and provide strong encapsulation.
