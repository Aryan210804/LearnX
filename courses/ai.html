Introduction to Artificial Intelligence

AI is the simulation of human intelligence by machines. It encompasses everything from simple rule-based algorithms to complex neural networks that can generate art, diagnose diseases, or drive cars.

The Core Hierarchy:
  • Artificial Intelligence (AI): The broad goal of creating systems capable of human-like tasks.
  • Machine Learning (ML): A subset of AI focused on algorithms that learn from data.
  • Deep Learning (DL): A specific ML technique using multi-layered neural networks.

How Machines "Learn"

In traditional software engineering, we define Rules + Data = Output. In Machine Learning, we flip the script: Data + Output = Rules (the Model). The computer finds the mathematical patterns itself.

Fun Fact

The term "Machine Learning" was coined in 1959 by Arthur Samuel. He created a program that learned to play checkers better than he could, proving machines aren't just limited to what we specifically tell them to do.

Types of Learning

Not all ML is trained the same way. We categorize algorithms based on their training style and data feedback.

Supervised Learning

Learning with labeled data. You provide inputs and the "correct answers."

  • Classification: Is this email spam? (Yes/No)
  • Regression: How much is this house worth? ($)

Unsupervised Learning

Finding patterns in unlabeled data. The machine groups data based on similarities.

  • Clustering: Grouping customers by shopping habits.
  • Association: People who buy milk also buy bread.

Reinforcement Learning (RL)

Learning by interaction—like training a dog. The agent receives "rewards" for correct actions and "penalties" for mistakes. This is how AlphaGo and OpenAI Five mastered complex strategy games.

The Machine Learning Workflow

Building AI is a scientific process. Skipping steps like data cleaning often results in poor model performance.

1. Problem Definition & Data Collection

Identify what you want to predict and gather raw data (CSV, SQL, API).

2. Data Preprocessing & Cleaning

Handling missing values, removing outliers, and normalizing data scales.

3. Feature Engineering

Selecting or creating the most relevant variables for prediction.

4. Model Training & Evaluation

Fitting the algorithm to data and testing it on "hidden" test sets to verify accuracy.

Regression Models

Regression is used when we want to predict a continuous numerical value. It is the backbone of financial forecasting, weather prediction, and scientific analysis.

Linear Regression

Predicts a dependent variable based on one or more independent variables. The goal is to find the "Line of Best Fit."

Formula: Y = mX + b

Logistic Regression

Despite the name, this is used for Classification. It predicts the probability of an input belonging to a certain class using a Sigmoid function.

Python: Linear Regression 
from sklearn.linear_model import LinearRegression
import numpy as np

# Training Data (X = SqFt, y = Price)
X = np.array([[1000], [1500], [2000], [2500]])
y = np.array([300000, 450000, 600000, 750000])

model = LinearRegression()
model.fit(X, y)
print(f"Price for 1800 sqft: ${model.predict([[1800]])[0]}")

Classification Models

Classification is used when the output variable is a category (e.g., "Red" or "Blue", "Spam" or "Not Spam").

K-Nearest Neighbors (KNN)

Classifies a data point based on how its neighbors are classified. If K=3, it looks at the 3 closest points.

Decision Trees

A flowchart-like structure where each internal node represents a "test" on an attribute.

Naive Bayes & SVM

Naive Bayes is based on Bayes' Theorem and is popular for text classification (like spam detection). Support Vector Machines (SVM) find the "hyperplane" that best separates different classes in high-dimensional space.

Clustering & Dimensionality Reduction

When we don't have labels, we use **Unsupervised Learning** to find hidden structures in the data.

K-Means Clustering

Divides data into 'K' number of clusters. Each point is assigned to the cluster with the nearest mean (centroid).

from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=3)
kmeans.fit(data)

Dimensionality Reduction (PCA)

Principal Component Analysis (PCA) reduces the number of variables in a dataset while preserving as much info as possible. This is vital for visualizing complex data and speeding up model training.

Perceptrons & Neural Network Basics

A Perceptron is the simplest form of a neural network—it simulates a single biological neuron. It takes multiple inputs, applies weights to them, and passes them through an activation function.

graph LR I1(Input 1) -- W1 --> S(Sum) I2(Input 2) -- W2 --> S S -- Activation --> O(Output)
Key Terminologies:
  • Weights: How much importance we give to an input.
  • Bias: An extra parameter to shift the activation function.
  • Activation Function: Determines if the neuron should "fire" (e.g., Sigmoid, ReLU).

Deep Learning Intro

Deep Learning is a specialized field of Machine Learning that uses Multi-Layered Neural Networks (Deep Neural Networks) to learn from vast amounts of data.

Why "Deep"?

The "Deep" refers to the number of layers. While a basic Perceptron has one layer, Deep Learning models can have hundreds of hidden layers, allowing them to learn incredibly complex patterns.

Modern Applications:
  • Computer Vision: Facial recognition, medical imaging.
  • Natural Language Processing: Large Language Models (LLMs) like GPT.
  • Generative AI: Creating images, videos, and music from prompts.

Example: Simple ANN (Python/Keras)

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

model = Sequential([
    Dense(16, activation='relu', input_shape=(10,)),
    Dense(8, activation='relu'),
    Dense(1, activation='sigmoid')
])

model.compile(optimizer='adam', loss='binary_crossentropy')

Brain.js Mastery

Brain.js is a GPU-accelerated library for Neural Networks in JavaScript. It’s perfect for developers who want to add high-level intelligence to their apps without learning complex math.

Key Features:
  • Easy to use: Train a network in under 10 lines of code.
  • Nueron-to-Neuron: Great for simple pattern recognition.
  • Browser & Node: Works anywhere JS runs.
Brain.js Training 
const net = new brain.NeuralNetwork();

// Input: RGB Colors, Output: Brightness
net.train([
  { input: { r: 0.03, g: 0.7, b: 0.5 }, output: { black: 1 } },
  { input: { r: 0.16, g: 0.09, b: 0.2 }, output: { white: 1 } }
]);

const output = net.run({ r: 1, g: 0.4, b: 0 }); // { white: 0.99, black: 0.002 }

TensorFlow.js Pro

Developed by Google, TensorFlow.js is the heavyweight champion of Web ML. It allows you to run production-grade Deep Learning models in the browser using WebGL/WebGPU.

Pre-trained Models

Use powerful models for face detection, pose estimation, and speech recognition with zero training required.

Transfer Learning

Take a massive model (like MobileNet) and retrain just the last layer to recognize your custom data.

Project Lab: House Price Predictor

Let's build a real Regression model to predict house prices based on Square Footage and Bedrooms.

Python Lab 
import pandas as pd
from sklearn.linear_model import LinearRegression

# 1. Dataset
data = {'SqFt': [1000, 1500, 2000, 2500, 3000],
        'Beds': [1, 2, 3, 3, 4],
        'Price': [250000, 375000, 500000, 550000, 700000]}
df = pd.DataFrame(data)

# 2. Train Model
X = df[['SqFt', 'Beds']]
y = df['Price']
model = LinearRegression().fit(X, y)

# 3. Predict
print(f"Prediction: ${model.predict([[1600, 2]])[0]:,.2f}")

AI Ethics & The Future

With great power comes great responsibility. As AI becomes more integrated into society, we must address critical ethical concerns.

Ethics Checklist:
  • Bias: Does the training data reflect diversity or reinforce stereotypes?
  • Privacy: Are we respecting users' data rights?
  • Safety: Can the model be manipulated by "Adversarial Attacks"?
  • Job Displacement: How can we reskill the workforce for an AI-driven economy?
