    CSS Masterclass

    CSS Introduction

    📌 Introduction

    CSS (Cascading Style Sheets) is the language we use to style an HTML document. CSS describes how HTML elements should be displayed.

    💻 Code Example

    body {
  background-color: lightblue;
}
h1 {
  color: white;
  text-align: center;
}

    CSS Syntax

    A CSS rule-set consists of a selector and a declaration block.

    h1 {color: blue; font-size: 12px;}
    • Selector: Points to the HTML element you want to style.
    • Declaration: Contains one or more declarations separated by semicolons.
    • Property: The style attribute you want to change.
    • Value: The value you want to give the property.

    CSS Selectors

    CSS selectors are used to "find" (or select) the HTML elements you want to style.

    • Element Selector: p { ... } (Selects all <p> elements)
    • Id Selector: #para1 { ... } (Selects element with id="para1")
    • Class Selector: .center { ... } (Selects elements with class="center")
    • Universal Selector: * { ... } (Selects all elements)
    • Grouping Selector: h1, h2, p { ... } (Selects multiple elements)

    CSS Comments

    Comments are used to explain the code, and may help when you edit the source code at a later date. Comments are ignored by browsers.

    /* This is a single-line comment */
p {
  color: red;
}
/* This is
a multi-line
comment */

    CSS Colors

    Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

    h1 {background-color: DodgerBlue;}
p {color: Tomato;}

    CSS Backgrounds

    CSS background properties are used to define the background effects for elements.

    • background-color
    • background-image
    • background-repeat
    • background-attachment
    • background-position

    CSS Borders

    The CSS border properties allow you to specify the style, width, and color of an element's border.

    p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}

    CSS Margins

    Margins are used to create space around elements, outside of any defined borders.

    div {
  margin: 70px;
  border: 1px solid #4CAF50;
}

    CSS Padding

    Padding is used to create space around an element's content, inside of any defined borders.

    div {
  padding: 70px;
  border: 1px solid #4CAF50;
}

    CSS Height and Width

    The height and width properties are used to set the height and width of an element.

    div {
  height: 200px;
  width: 50%;
  background-color: powderblue;
}

    The Box Model

    📌 Introduction

    Every element is a rectangular box containing Content, Padding, Border, and Margin.

    📊 Visual Concept

    Compare the logic below:

    graph TD Content --> Padding Padding --> Border Border --> Margin

    💻 Code Examples

    div {
  width: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

    ⚡ Best Practices

    • ✅ Use box-sizing: border-box to make math easier.

    CSS Outline

    An outline is a line drawn outside the element's border.

    p {
  border: 1px solid black;
  outline: 5px dotted red;
}

    CSS Text

    Text properties: text-align, text-decoration, text-transform, letter-spacing, line-height.

    CSS Fonts

    Font properties define the font family, boldness, size, and the style of a text.

    p {
  font-family: "Times New Roman", Times, serif;
  font-size: 30px;
}

    CSS Links

    Links can be styled depending on what state they are in. The four links states are: a:link, a:visited, a:hover, a:active.

    CSS Lists

    In HTML, there are two main types of lists: unordered and ordered. CSS list properties allow you to change diffrent list item markers, set an image as the list item marker, etc.

    CSS Tables

    To specify table borders in CSS, use the border property.

    table, th, td {
  border: 1px solid black;
}

    CSS Display

    The display property determines how an element is displayed.

    • display: block; (Takes full width)
    • display: inline; (Takes necessary width)
    • display: none; (Hidden)

    CSS Position

    The position property specifies the type of positioning method used for an element.

    • static (Default)
    • relative
    • absolute
    • fixed
    • sticky

    CSS Rounded Corners

    The border-radius property defines the radius of the element's corners.

    #rcorners1 {
  border-radius: 25px;
  background: #73AD21;
  padding: 20px;
  width: 200px;
  height: 150px;
}

    CSS Gradients

    CSS gradients let you display smooth transitions between two or more specified colors.

    Linear Gradient

    #grad {
  background-image: linear-gradient(red, yellow);
}

    Radial Gradient

    #grad {
  background-image: radial-gradient(red, yellow, green);
}

    CSS Shadows

    With CSS you can add shadow to text and to elements.

    h1 {
  text-shadow: 2px 2px 5px red;
}
div {
  box-shadow: 10px 10px 5px grey;
}

    CSS 2D Transforms

    CSS transforms allow you to move, rotate, scale, and skew elements.

    div {
  transform: translate(50px, 100px);
  transform: rotate(20deg);
  transform: scale(2, 3);
  transform: skewX(20deg);
}

    CSS 3D Transforms

    CSS allows you to format your elements using 3D transforms.

    div {
  transform: rotateX(150deg);
}

    CSS Transitions

    CSS transitions allows you to change property values smoothly, over a given duration.

    div {
  width: 100px;
  height: 100px;
  background: red;
  transition: width 2s;
}
div:hover {
  width: 300px;
}

    CSS Animations

    CSS allows animation of HTML elements without using JavaScript or Flash!

    @keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

div {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: example;
  animation-duration: 4s;
}

    CSS Tooltips

    A tooltip is often used to specify extra information about something when the user moves the mouse pointer over an element.

    CSS Image Styling

    Learn to style images using CSS.

    img {
  border-radius: 8px;
  opacity: 0.5;
}

    CSS Buttons

    Learn how to style buttons with CSS.

    .button {
  background-color: #4CAF50;
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
}

    CSS Pagination

    If you have a web site with lots of pages, you may wish to add some sort of pagination to each page.

    CSS Media Queries

    Media query is a CSS technique introduced in CSS3. It uses the @media rule to include a block of CSS properties only if a certain condition is true.

    @media only screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

    CSS Flexbox

    📌 Introduction

    The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

    .flex-container {
  display: flex;
  background-color: DodgerBlue;
}

.flex-container > div {
  background-color: #f1f1f1;
  margin: 10px;
  padding: 20px;
  font-size: 30px;
}

    Parent Properties (Container)

    • flex-direction (row, column)
    • flex-wrap (wrap, nowrap)
    • justify-content (center, space-between)
    • align-items (center, stretch)

    CSS Grid

    📌 Introduction

    The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

    .grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
  background-color: #2196F3;
  padding: 10px;
}
.grid-item {
  background-color: rgba(255, 255, 255, 0.8);
  border: 1px solid rgba(0, 0, 0, 0.8);
  padding: 20px;
  font-size: 30px;
  text-align: center;
}

    Grid Properties

    • grid-template-columns
    • grid-template-rows
    • grid-gap

    Responsive Web Design

    Responsive Web Design is about creating web pages that look good on all devices! It is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones).

    The Viewport

    The viewport is the user's visible area of a web page.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    Media Queries

    /* On screens that are 600px or less, set the background color to olive */
@media screen and (max-width: 600px) {
  body {
    background-color: olive;
  }
}

    SASS Tutorial

    📌 What is Sass?

    Sass (Syntactically Awesome Style Sheets) is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS).

    Variables

    $font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
  font: 100% $font-stack;
  color: $primary-color;
}

    Nesting

    nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }
  li { display: inline-block; }
  a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
  }
}
