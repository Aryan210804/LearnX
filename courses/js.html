Intro to JavaScript

Chapter 1
Previous Next

📌 What is JavaScript?

JavaScript (JS) is the programming language of the web. While HTML is the structure and CSS is the style, JavaScript is the brain that makes things happen.

🚀 Capabilities

  • Change HTML Content (innerHTML).
  • Change HTML Styles (style.fontSize).
  • Hide/Show Elements.
  • Fetch data from servers.

💻 Outputting Data

There are 4 ways to output data in JS:

innerHTML = "Hello";      // Write to HTML element
document.write("Hello");  // Write to HTML output
window.alert("Hello");    // Popup alert box
console.log("Hello");     // Write to browser console

Variables & Data Types

Chapter 2
Previous Next

📌 Variables

Variables are containers for storing data.

  • const: Cannot be reassigned (Use this by default).
  • let: Can be reassigned (Use for loops or changing values).
  • var: Old school (Avoid using this).
const pi = 3.14;
let age = 25;
age = 26; // Allowed
Live Editor

📦 Data Types

  • String: "Hello"
  • Number: 42 or 3.14
  • Boolean: true or false
  • Object: {name:"John", age:30}
  • Array: ["Apple", "Banana"]
  • Null / Undefined: Empty or no value.

Operators

Chapter 3
Previous Next

Operators perform operations on variables and values.

  • Arithmetic: +, -, *, /, % (Modulus)
  • Assignment: =, +=, -=
  • Comparison: == (value), === (value & type), !=, >, <
  • Logical: && (AND), || (OR), ! (NOT)
let sum = 10 + 5; // 15
let isTrue = (10 > 5) && (5 < 10); // true

📌 Conditional Logic

Code that runs only if a condition is met.

📊 Visual Concept

graph TD Start --> Condition{Is it Raining?} Condition -- Yes --> Umbrella Condition -- No --> Sunglasses Umbrella --> Go Sunglasses --> Go

💻 Code

if (hour < 12) {
  greeting = "Good Morning";
} else if (hour < 18) {
  greeting = "Good Afternoon";
} else {
  greeting = "Good Evening";
}

🔀 Switch Statement

Use switch to select one of many blocks to execute.

switch(day) {
  case 0: day = "Sunday"; break;
  case 1: day = "Monday"; break;
  default: day = "Weekday";
}

📌 Lists

An array is a special variable, which can hold more than one value.

Common Methods

  • push(): Add new item to end.
  • pop(): Remove last item.
  • map(): Create new array by transforming every item.
  • filter(): Create new array with items that pass a test.
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2); // [2, 4, 6]
