    JavaScript Masterclass

    Intro to JavaScript

    📌 What is JavaScript?

    JavaScript (JS) is the programming language of the web. While HTML is the structure and CSS is the style, JavaScript is the brain that makes things happen.

    🚀 Capabilities

    • Change HTML Content (innerHTML).
    • Change HTML Styles (style.fontSize).
    • Hide/Show Elements.
    • Fetch data from servers.

    💻 Outputting Data

    There are 4 ways to output data in JS:

    innerHTML = "Hello";      // Write to HTML element
document.write("Hello");  // Write to HTML output
window.alert("Hello");    // Popup alert box
console.log("Hello");     // Write to browser console

    Variables & Data Types

    📌 Variables

    Variables are containers for storing data.

    • const: Cannot be reassigned (Use this by default).
    • let: Can be reassigned (Use for loops or changing values).
    • var: Old school (Avoid using this).
    const pi = 3.14;
let age = 25;
age = 26; // Allowed

    📦 Data Types

    • String: "Hello"
    • Number: 42 or 3.14
    • Boolean: true or false
    • Object: {name:"John", age:30}
    • Array: ["Apple", "Banana"]
    • Null / Undefined: Empty or no value.

    Operators

    Operators perform operations on variables and values.

    • Arithmetic: +, -, *, /, % (Modulus)
    • Assignment: =, +=, -=
    • Comparison: == (value), === (value & type), !=, >, <
    • Logical: && (AND), || (OR), ! (NOT)
    let sum = 10 + 5; // 15
let isTrue = (10 > 5) && (5 < 10); // true

    Control Flow (If/Else)

    📌 Conditional Logic

    Code that runs only if a condition is met.

    📊 Visual Concept

    graph TD Start --> Condition{Is it Raining?} Condition -- Yes --> Umbrella Condition -- No --> Sunglasses Umbrella --> Go Sunglasses --> Go

    💻 Code

    if (hour < 12) {
  greeting = "Good Morning";
} else if (hour < 18) {
  greeting = "Good Afternoon";
} else {
  greeting = "Good Evening";
}

    🔀 Switch Statement

    Use switch to select one of many blocks to execute.

    switch(day) {
  case 0: day = "Sunday"; break;
  case 1: day = "Monday"; break;
  default: day = "Weekday";
}

    Loops

    📌 Iteration

    Loops can execute a block of code a number of times.

    For Loop

    for (let i = 0; i < 5; i++) {
  console.log("Count: " + i);
}

    While Loop

    while (condition) {
  // code block
}

    For..Of (Arrays)

    const cars = ["BMW", "Volvo", "Mini"];
for (let car of cars) {
  console.log(car);
}

    Functions

    📌 Reusable Code

    Functions are blocks of code designed to perform a particular task.

    Function Declaration

    function greet(name) {
  return "Hello " + name;
}

    Arrow Functions (ES6)

    const greet = (name) => "Hello " + name;

    Objects

    📌 Real Life Objects

    In real life, a car is an object. It has properties like weight and color, and methods like start and stop. 

    const car = {
  type: "Fiat",
  model: "500",
  color: "white",
  start: function() {
    console.log("Vroom!");
  }
};

console.log(car.model); // Dot notation

    Arrays & Methods

    📌 Lists

    An array is a special variable, which can hold more than one value.

    const fruits = ["Banana", "Orange", "Apple"];

    Common Methods

    • push(): Add new item to end.
    • pop(): Remove last item.
    • map(): Create new array by transforming every item.
    • filter(): Create new array with items that pass a test.
    const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2); // [2, 4, 6]

    String Methods

    JavaScript strings are for storing and manipulating text.

    • length: Returns length of string.
    • slice(start, end): Extracts a part of a string.
    • replace("old", "new"): Replaces content.
    • toUpperCase(): Converts to Upper Case.

    Math & Dates

    Math Object

    Math.round(4.6); // 5
Math.random();   // 0.293..

    Date Object

    const d = new Date();
console.log(d.getFullYear());

    DOM Manipulation

    📌 The Document Object Model

    The DOM is the tree structure of your HTML. JS can change the DOM to update the page.

    Selecting Elements

    document.getElementById("demo");
document.querySelector(".myClass");

    Changing Content

    document.getElementById("header").innerHTML = "New Title";
document.getElementById("img").src = "new.jpg";
document.getElementById("p1").style.color = "blue";

    Events

    HTML events are "things" that happen to HTML elements.

    • onclick: User clicks an element.
    • onmouseover: Mouse is over an element.
    • onkeydown: Key is pressed.
    btn.addEventListener("click", function() {
  alert("Button Clicked!");
});

    Async JS & Fetch API

    📌 Asynchronous JavaScript

    Functions running in parallel with other functions.

    Fetch API

    async function getData() {
  const response = await fetch("https://api.example.com/data");
  const data = await response.json();
  console.log(data);
}

    ES6+ Features

    Destructuring

    const [a, b] = [1, 2];
const {name, age} = person;

    Spread Operator

    const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // [1, 2, 3, 4]

    Template Literals

    const msg = `Hello ${name}`;
