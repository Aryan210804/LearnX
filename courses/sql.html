    Introduction to SQL

    📌 What is SQL?

    SQL (Structured Query Language) is the standard language for dealing with Relational Databases.

    graph LR User --> SQL_Query SQL_Query --> Database Database --> Result_Data

    Select Statement

    The SELECT statement is used to select data from a database.

    SELECT * FROM Customers;
    SELECT FirstName, LastName FROM Employees;

    Where Clause

    Filter records.

    SELECT * FROM Customers
WHERE Country = 'Mexico';

    Insert Into

    Adding new records.

    INSERT INTO Customers (CustomerName, City, Country)
VALUES ('Cardinal', 'Stavanger', 'Norway');

    Update

    Modify existing records.

    UPDATE Customers
SET ContactName = 'Alfred Schmidt', City = 'Frankfurt'
WHERE CustomerID = 1;

    Delete

    Remove records.

    DELETE FROM Customers WHERE CustomerName='Alfreds Futterkiste';

    SQL Functions

    MIN(), MAX(), COUNT(), AVG(), SUM().

    SELECT COUNT(ProductID)
FROM Products;

    Wildcards

    Pattern matching.

    SELECT * FROM Customers
WHERE City LIKE 'ber%'; -- Starts with "ber"

    Joins

    Combine rows from two or more tables.

    graph TD Table1 --> Join Table2 --> Join Join --> Result 
    SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;

    Union

    Combine the result-set of two or more SELECT statements.

    Group By

    Group rows that have the same values into summary rows.

    SELECT COUNT(CustomerID), Country
FROM Customers
GROUP BY Country;

    Create DB/Table

    CREATE DATABASE testDB;
CREATE TABLE Persons (
    PersonID int,
    LastName varchar(255),
    FirstName varchar(255),
    Address varchar(255),
    City varchar(255)
);

    Constraints

    Rules for data.

    • NOT NULL
    • UNIQUE
    • PRIMARY KEY
    • FOREIGN KEY

    Normalization

    Organizing data to reduce redundancy (1NF, 2NF, 3NF).
