    Introduction to R

    📌 What is R?

    R is a language and environment for statistical computing and graphics.

    graph TD Data --> Statistics Statistics --> Visualization Visualization --> Insights

    Syntax

    Assignment uses <-.

    name <- "John"
age <- 25
print(name)

    Data Types

    • Numeric: 10.5
    • Integer: 10L
    • Character: "Hello"
    • Logical: TRUE

    Vectors

    A list of items of the same type.

    fruits <- c("apple", "banana", "cherry")

    Lists

    A collection of data which is ordered and changeable.

    thislist <- list("apple", "banana", "cherry")

    Matrices

    A 2D data set with columns and rows.

    thismatrix <- matrix(c(1,2,3,4,5,6), nrow = 3, ncol = 2)

    Data Frames

    Data tables. similar to Pandas DataFrames.

    Data_Frame <- data.frame (
  Training = c("Strength", "Stamina", "Other"),
  Pulse = c(100, 150, 120),
  Duration = c(60, 30, 45)
)

    Statistics

    R is built for stats.

    mean(Data_Frame$Pulse)
median(Data_Frame$Pulse)
min(Data_Frame$Pulse)

    Plotting

    Simple plotting.

    plot(1:10)

    GGPlot2

    Advanced visualization.

    library(ggplot2)
ggplot(df, aes(x=dose, y=len)) + geom_point()
