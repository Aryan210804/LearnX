    Introduction to Python

    📌 Introduction

    Python is defined by its simplicity. It prioritizes readability, making it the perfect first language, yet it powers NASA, Google, and OpenAI.

    🚀 Why Python?

    • Readable: Syntax like English.
    • Versatile: Web, AI, Data Science, Automation.
    • Huge Community: Libraries for everything.

    Variables & Data Types

    📌 Variables

    No command for declaring a variable. A variable is created the moment you first assign a value to it.

    x = 5
y = "John"
print(x)
print(y)

    📦 Core Data Types

    • Text Type: str
    • Numeric Types: int, float, complex
    • Sequence Types: list, tuple, range
    • Mapping Type: dict
    • Set Types: set, frozenset
    • Boolean Type: bool

    Lists & Tuples

    📌 Lists

    Lists are used to store multiple items in a single variable. Ordered, Changeable.

    mylist = ["apple", "banana", "cherry"]
mylist.append("orange")
print(mylist[1]) # banana

    📌 Tuples

    Ordered, Unchangeable.

    mytuple = ("apple", "banana", "cherry")

    Sets

    Unordered, Unindexed, No Duplicates.

    myset = {"apple", "banana", "cherry"}

    Dictionaries

    📌 Key-Value Pairs

    Dictionaries are used to store data values in key:value pairs.

    thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(thisdict["brand"])

    Conditionals

    Python relies on indentation (whitespace) to define scope.

    a = 33
b = 200
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")

    Loops

    While Loop

    i = 1
while i < 6:
  print(i)
  i += 1

    For Loop

    fruits = ["apple", "banana", "cherry"]
for x in fruits:
  print(x)

    Classes & OOP

    📌 Classes/Objects

    Python is an object oriented programming language.

    class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("John", 36)
print(p1.name)

    File Handling

    The key function for working with files in Python is the open() function.

    f = open("demofile.txt", "r")
print(f.read())

    Exception Handling

    When an error occurs, or exception as we call it, Python will normally stop and generate an error message. 

    try:
  print(x)
except:
  print("An exception occurred")
