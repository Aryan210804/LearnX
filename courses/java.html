    Java Introduction

    📌 What is Java?

    Java is a popular programming language, created in 1995. It is owned by Oracle, and more than 3 billion devices run Java.

    🚀 Main Features

    • Object-Oriented: Everything is an Object.
    • Platform Independent: "Write Once, Run Anywhere".
    • Secure: Robust security features.

    💻 Syntax

    public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

    Variables & Data Types

    📌 Variables

    Containers for storing data values.

    String name = "John";
int myNum = 15;
final int myNum = 15; // Final (constant)

    📦 Data Types

    • int: Integers (whole numbers).
    • float / double: Floating point numbers.
    • char: Single characters ('a').
    • boolean: true or false.
    • String: Text.

    Operators & Casting

    Operators

    +, -, *, /, % (Arithmetic)

    ==, !=, >, < (Comparison)

    Type Casting

    Widening: int -> double (Automatic)

    Narrowing: double -> int (Manual)

    double myDouble = 9.78d;
int myInt = (int) myDouble; // 9

    Conditionals

    Java supports logical conditions from mathematics.

    if (time < 18) {
  System.out.println("Good day.");
} else {
  System.out.println("Good evening.");
}

    Switch Statement

    Use the switch statement to select one of many code blocks to be executed.

    switch(day) {
  case 1:
    System.out.println("Monday");
    break;
  case 2:
    System.out.println("Tuesday");
    break;
}

    Loops

    While Loop

    while (i < 5) {
  System.out.println(i);
  i++;
}

    For Loop

    for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

    For-Each Loop

    for (String i : cars) {
  System.out.println(i);
}

    Methods

    A method is a block of code which only runs when it is called.

    static void myMethod(String fname) {
  System.out.println(fname + " Refsnes");
}

    Arrays

    Arrays are used to store multiple values in a single variable.

    String[] cars = {"Volvo", "BMW", "Ford"};
System.out.println(cars[0]); // Outputs Volvo

    Encapsulation

    The meaning of Encapsulation, is to make sure that "sensitive" data is hidden from users.

    Declare class variables/attributes as private and provide public get and set methods to access and update the value of a private variable.

    Inheritance

    In Java, it is possible to inherit attributes and methods from one class to another.

    class Vehicle {
  protected String brand = "Ford";
  public void honk() {
    System.out.println("Tuut, tuut!");
  }
}

class Car extends Vehicle {
  private String modelName = "Mustang";
}

    Polymorphism

    Polymorphism means "many forms", and it occurs when we have many classes that are related to each other by inheritance.

    Abstraction

    Data abstraction is the process of hiding certain details and showing only essential information to the user.

    • Abstract Class: Cannot be used to create objects.
    • Interface: A completely "abstract class" that finds grouping with unrelated classes.

    Java Files

    The File class from the java.io package, allows us to work with files.

    import java.io.File; 
File myObj = new File("filename.txt");

    Exception Handling

    The try statement allows you to define a block of code to be tested for errors.

    The catch statement allows you to define a block of code to be executed, if an error occurs in the try block.

    try {
  int[] myNumbers = {1, 2, 3};
  System.out.println(myNumbers[10]);
} catch (Exception e) {
  System.out.println("Something went wrong.");
}
