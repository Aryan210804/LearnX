    Introduction to C

    📌 What is C?

    C is a general-purpose programming language tailored for systems programming (like OS).

    graph TD Source_Code --> Compiler Compiler --> Machine_Code Machine_Code --> Execution

    Syntax

    #include <stdio.h>

int main() {
  printf("Hello World!");
  return 0;
}

    Variables

    Containers for storing data values.

    int myNum = 15;
float myFloatNum = 5.99;
char myLetter = 'D';

    Data Types

    • int: Integers
    • float: Floating point numbers
    • char: Single character

    If...Else

    if (20 > 18) {
  printf("20 is greater than 18");
}

    Switch

    Select one of many code blocks to be executed.

    Loops

    int i;
for (i = 0; i < 5; i++) {
  printf("%d\n", i);
}

    Functions

    void myFunction() {
  printf("I just got executed!");
}

    Arrays

    int myNumbers[] = {25, 50, 75, 100};

    Pointers

    A variable that stores the memory address of another variable.

    int myAge = 43;
int* ptr = &myAge;

printf("%p", ptr); // Outputs memory address

    Memory Management

    malloc(), calloc(), free().
