Complete Excel Tutorial

Learn Excel from Basics to Advanced

Excel HOME

Definition: The Excel Home tab provides core functionalities like copy, paste, font, alignment, number formats, and styles.

Example: Copy cells A1 to A3 and paste in B1 to B3.

Excel Introduction

Definition: Microsoft Excel is a spreadsheet application used for data entry, analysis, visualization, and calculation.

Syntax: Use cells (like A1) to store data. Functions like =SUM(A1:A3) perform calculations.

=SUM(A1:A5)
=AVERAGE(B1:B3)

Excel Get Started

Definition: Getting started includes launching Excel, opening a workbook, and understanding the user interface.

Syntax: N/A (interface-based)

Open Excel > File > New Workbook

Excel Syntax

Definition: Excel formulas begin with = followed by the function name and parameters.

=FUNCTION(arguments)
Example:
=SUM(A1:A3)

Excel Ranges

Definition: A range is a group of cells selected for performing operations.

A1:A10 (vertical range)
A1:D1 (horizontal range)

Excel Fill

Definition: Fill feature is used to auto-complete a series like numbers, dates, or formulas.

Enter 1 in A1, drag down to auto-fill 2, 3, ...

Excel Formulas

Definition: A formula is an expression that calculates the value of a cell.

=A1+B1
=A1*B1

Excel Functions

Definition: Functions are predefined formulas to perform calculations.

=SUM(A1:A5)
=IF(B1>100, "Yes", "No")

Excel Formatting

Definition: Formatting refers to changing the appearance of data using font, color, borders, number styles etc.

Change cell B1 to bold and red color

Excel Charts

Definition: Charts provide a visual representation of data using bars, lines, pies etc.

Select data > Insert > Chart > Column

Excel Tables

Definition: Excel Tables organize data in a structured format with filtering, sorting, and styling built-in.

Select range > Insert > Table > Confirm headers

Excel Conditional Formatting

Definition: Automatically format cells based on rules or conditions.

Home > Conditional Formatting > Highlight Cells Rules > Greater Than > 100

Highlight Cell Rules

Definition: Predefined rules to highlight cells meeting specific criteria (e.g., values, duplicates).

Highlight all cells greater than 500

Top Bottom Rules

Definition: Format top or bottom percentage or number of cells.

Highlight top 10 items in sales data

Data Bars

Definition: Adds horizontal bars inside cells to visualize data magnitude.

Select cells > Conditional Formatting > Data Bars > Choose style

Color Scales

Definition: Colors cells based on their values, creating a gradient effect.

Select data > Conditional Formatting > Color Scales > Select color scale

Icon Sets

Definition: Add icons (arrows, flags, stars) to cells based on value ranges.

Conditional Formatting > Icon Sets > Choose icons

Manage Rules (CF)

Definition: Manage, edit, or delete existing conditional formatting rules.

Home > Conditional Formatting > Manage Rules > Edit or Delete rules

Table Pivot Intro

Definition: Pivot Tables summarize large data sets quickly and flexibly.

Select data > Insert > Pivot Table > Choose fields to summarize

Create a sales sheet with product names, quantities, prices, and calculate totals

Case: Poke Mart, Styling

Definition: Applying formatting and styles to the Poke Mart sales sheet for clarity and presentation.

Apply bold headers, currency formatting, and conditional formatting for low stock

Functions List

Below are some common Excel functions with syntax and examples:

  • AND: =AND(logical1, logical2, ...) - Returns TRUE if all arguments are TRUE.
  • AVERAGE: =AVERAGE(range) - Returns average of numbers in range.
  • IF: =IF(condition, value_if_true, value_if_false) - Conditional logic.
  • VLOOKUP: =VLOOKUP(lookup_value, table_array, col_index, [range_lookup]) - Lookup value vertically.
  • SUM: =SUM(range) - Adds all numbers in range.
  • COUNT: =COUNT(range) - Counts numeric cells in range.
  • LEN: =LEN(text) - Counts number of characters.
  • NOW: =NOW() - Current date and time.
  • CONCATENATE (or CONCAT): =CONCATENATE(text1, text2,...) - Joins text.
