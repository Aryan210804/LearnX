C Intro

C is a general-purpose, procedural programming language developed in the 1970s. It is widely used for system and application software.

Get Started

To start programming in C, you need a compiler like GCC. Save your code in a file with .c extension and compile it using the compiler.

gcc filename.c -o outputname

C Syntax

Basic structure of a C program.

#include <stdio.h>
int main() {
  printf("Hello, World!\\n");
  return 0;
}
Hello, World!

Output

Use printf() to display output on the screen.

printf("Your message here");

Comments

Comments are non-executable code used to explain the program.

// Single line comment
/* Multi-line
comment */

Variables

Variables store data values.

int age = 25;
float height = 5.9;

Data Types

Defines the type of data a variable can hold.

int, float, double, char, void

Constants

Fixed values that do not change during program execution.

#define PI 3.14

Operators

Symbols that perform operations on variables and values.

+, -, *, /, %, ++, --, ==, !=, <, >, <=, >=, &&, ||, !

Booleans

Boolean type stores TRUE (1) or FALSE (0) values.

_Bool flag = 1; // true

If...Else

Used for conditional branching.

if (age >= 18) {
  printf("Adult");
} else {
  printf("Minor");
}

Switch

Selects one of many code blocks to execute.

switch (choice) {
  case 1: printf("One"); break;
  case 2: printf("Two"); break;
  default: printf("Other");
}

While Loop

Repeats code while a condition is true.

int i = 0;
while (i < 5) {
  printf("%d\\n", i);
  i++;
}

For Loop

Repeats code a known number of times.

for (int i = 0; i < 5; i++) {
  printf("%d\\n", i);
}

Break / Continue

break exits the loop; continue skips to the next iteration.

for (int i = 0; i < 5; i++) {
  if (i == 3) break;
  if (i == 1) continue;
  printf("%d\\n", i);
}

Arrays

Collection of variables of the same type stored consecutively.

int numbers[5] = {1, 2, 3, 4, 5};

Strings

Array of characters ending with a null character '\0'.

char name[] = "Aryan";

User Input

Using scanf() to get input from the user.

int age;
scanf("%d", &age);

Memory Address

Use & to get the memory address of a variable.

int x = 5;
printf("%p", &x);

Pointers

Variables that store memory addresses.

int x = 5;
int *ptr = &x;
printf("%d", *ptr); // Output: 5

Functions

Reusable blocks of code performing specific tasks.

int add(int a, int b) {
  return a + b;
}

Function Parameters

Variables passed to functions to provide input.

void greet(char name[]) {
  printf("Hello, %s!\\n", name);
}

greet("Aryan"); // Output: Hello, Aryan!
Hello, Aryan!

Scope

Defines where a variable is accessible.

int x = 10; // Global
void func() {
  int x = 5; // Local
  printf("x = %d\\n", x);
}

func();
printf("x = %d\\n", x);
x = 5
x = 10

Function Declaration

Declare functions before use or provide prototypes.

int multiply(int, int); // Prototype

int main() {
  printf("%d\\n", multiply(3, 4));
  return 0;
}

int multiply(int a, int b) {
  return a * b;
}
12

Recursion

A function calling itself.

int factorial(int n) {
  if (n == 0) return 1;
  else return n * factorial(n - 1);
}

Math Functions

Use math.h for math functions.

#include <math.h>
double result = sqrt(16); // 4.0
result = pow(2, 3); // 8.0

Files

Create, write, and read files using fopen, fprintf, fscanf, and fclose.

// Create and Write
FILE *fp = fopen("file.txt", "w");
fprintf(fp, "Hello file!\\n");
fclose(fp);

// Read
char str[100];
fp = fopen("file.txt", "r");
fscanf(fp, "%[^\n]", str);
printf("%s\\n", str);
fclose(fp);
Hello file!

Structures

Group of variables under one name.

struct Person {
  char name[50];
  int age;
};

struct Person p1 = {"Aryan", 21};
printf("%s is %d years old.", p1.name, p1.age);

Unions

Similar to structures but members share same memory.

union Data {
  int i;
  float f;
  char str[20];
};

union Data data;
data.i = 10;
printf("%d", data.i);

Enums

User-defined types with named integer constants.

enum Color {RED, GREEN, BLUE};
enum Color c = RED;

Memory Management

Dynamic memory allocation with malloc, calloc, free.

int *ptr = (int *) malloc(sizeof(int));
*ptr = 10;
free(ptr);

Macros

Preprocessor directives for constants or functions.

#define PI 3.14
#define SQUARE(x) ((x) * (x))
