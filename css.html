CSS Tutorial

CSS HOME

Definition: CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout and look of elements on a webpage.

Example:

p {
  color: blue;
  font-size: 16px;
}

Output:

This is a blue paragraph with 16px font size.

CSS Introduction

Definition: CSS allows you to apply styles to web documents. More importantly, CSS enables you to do this independently of the HTML that makes up each web page.

Syntax:

selector {
  property: value;
}

Example:

body {
  background-color: lightyellow;
}

Output:

This area has a light yellow background using CSS.

CSS Syntax

Definition: A CSS rule consists of a selector and a declaration block. The selector points to the HTML element to style. The declaration block contains one or more declarations separated by semicolons.

Syntax:

selector {
  property: value;
}

Example:

h1 {
  color: red;
  text-align: center;
}

CSS Combinators

Definition: CSS combinators describe the relationship between selectors:

  • (descendant selector)
  • > (child selector)
  • + (adjacent sibling selector)
  • ~ (general sibling selector)

Example:

div p {
  color: blue; /* all p inside div */
}
div > p {
  color: red; /* direct child p of div */
}
h1 + p {
  font-weight: bold; /* p immediately after h1 */
}
h1 ~ p {
  font-style: italic; /* all p siblings after h1 */
}

Output:

This paragraph inside div is blue (descendant).

Another paragraph inside div is also blue.

But if it was direct child (use >), color would be red.

Paragraph immediately after h1 is bold.

Paragraphs after h1 are italic.

CSS Grid Columns/Rows

Definition: Define the number and size of columns and rows in a grid container.

Example:

.container {
  display: grid;
  grid-template-columns: 150px 1fr 2fr;
  grid-template-rows: 50px 100px;
}

CSS Grid Container

Definition: The element with display: grid; which establishes a grid formatting context for its children.

Key Properties: grid-template-columns, grid-template-rows, grid-gap, justify-items, align-items

CSS Grid Item

Definition: The direct children of a grid container which can be placed and sized within the grid using properties.

Key Properties: grid-column-start, grid-column-end, grid-row-start, grid-row-end, justify-self, align-self

Example:

.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}

CSS Responsive

Responsive Web Design (RWD) - Introduction

Responsive Web Design is an approach to web design that makes web pages render well on various devices and window or screen sizes. It uses flexible layouts, flexible images, and media queries.

RWD Viewport

The viewport tag controls the layout on mobile browsers. It ensures proper scaling and rendering on different screen sizes.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Responsive Grid View

Grid layout allows content to be divided into columns and rows, adapting to screen sizes easily.

.container { display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); gap: 10px; }

Media Queries

Media Queries apply different styles for different screen sizes and resolutions.

@media (max-width: 600px) { body { background-color: lightblue; } }

Responsive Images

Images can be made responsive using CSS. This ensures images fit within their containers.

img { max-width: 100%; height: auto; }

Responsive Videos

To make videos responsive, use a container with a relative aspect ratio.

.video-container { position: relative; padding-bottom: 56.25%; height: 0; } .video-container iframe { position: absolute; width: 100%; height: 100%; }

CSS SAAS

What is SASS?

SASS (Syntactically Awesome Stylesheets) is a CSS preprocessor that adds special features like variables, nesting, and functions.

SASS Variables

Variables store reusable values.

$main-color: #3498db; body { background-color: $main-color; }

SASS Nesting

You can nest selectors inside each other to show hierarchy.

nav { ul { list-style: none; } li { display: inline-block; } }

SASS Mixins

Mixins are reusable blocks of code.

@mixin center { display: flex; justify-content: center; align-items: center; } .box { @include center; }

Compiling SASS

SASS files must be compiled into regular CSS using tools like sass CLI or build tools.

// SCSS Input $color: red; h1 { color: $color; } // Compiled CSS Output h1 { color: red; }

CSS References

CSS Reference

The CSS Reference provides a comprehensive list of all CSS properties and their possible values. It's an essential resource for developers to understand how to style HTML elements effectively.

For a complete list, visit the W3Schools CSS Reference.

CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They can target elements based on name, id, class, attribute, and more.

/* Selects all <p> elements */ p { color: blue; } /* Selects element with id="header" */ #header { background-color: gray; } /* Selects all elements with class="intro" */ .intro { font-size: 1.2em; }

Explore more at the W3Schools CSS Selectors Reference.

CSS Combinators

Combinators define the relationship between selectors, allowing for more precise targeting of elements.

/* Descendant combinator */ div p { color: red; } /* Child combinator */ div > p { color: green; } /* Adjacent sibling combinator */ div + p { color: blue; } /* General sibling combinator */ div ~ p { color: purple; }

Learn more at the W3Schools CSS Combinators Reference.

CSS Pseudo-classes

Pseudo-classes allow you to define the special state of an element, such as when a user hovers over it or when it's the first child of its parent.

/* Hover state */ a:hover { color: red; } /* First child */ p:first-child { font-weight: bold; }

Detailed information can be found at the W3Schools CSS Pseudo-classes Reference.

CSS Pseudo-elements

Pseudo-elements allow you to style specific parts of an element, such as the first letter or line.

/* First letter */ p::first-letter { font-size: 2em; color: red; } /* Before content */ p::before { content: "Note: "; font-weight: bold; }

Explore more at the W3Schools CSS Pseudo-elements Reference.

CSS At-rules

At-rules are special statements that provide CSS with instructions, such as importing stylesheets or defining media queries.

/* Importing a stylesheet */ @import url("styles.css"); /* Media query */ @media (max-width: 600px) { body { background-color: lightgray; } }

Learn more at the W3Schools CSS At-rules Reference.

CSS Functions

CSS functions are used to perform calculations or manipulate values within CSS properties.

/* Using calc() */ div { width: calc(100% - 50px); } /* Using var() */ :root { --main-color: #3498db; } p { color: var(--main-color); }

Explore more at the W3Schools CSS Functions Reference.

CSS Aural

CSS Aural properties are used to define how content should be presented using speech synthesis, enhancing accessibility for users with visual impairments.

/* Example of aural properties */ p { azimuth: center-left; elevation: above; pause-before: 20ms; pitch: high; }

Learn more at the W3Schools CSS Aural Reference.
