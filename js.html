JavaScript Asynchronous Programming

Callbacks

Callbacks are functions passed as arguments and executed later.

function greet(name, callback) {
  console.log("Hi " + name);
  callback();
}
greet("John", function() {
  console.log("Callback executed.");
});

Promises

Promises represent future completion or failure of an asynchronous operation.

let promise = new Promise((resolve, reject) => {
  let success = true;
  if (success) resolve("Success!");
  else reject("Failure!");
});
promise.then(alert).catch(alert);

Async / Await

async and await simplify working with promises.

async function fetchData() {
  let response = await fetch("https://api.example.com/data");
  let data = await response.json();
  console.log(data);
}

document.getElementById("btn").onclick = function() {
  alert("Button clicked!");
};

DOM Event Listener

Event listeners listen for specified events and execute a callback when triggered.

document.getElementById("btn").addEventListener("click", function() {
  alert("Event Listener triggered!");
});

DOM Navigation

DOM navigation allows access to different parts of the document.

  • parentNode
  • childNodes
  • firstChild
  • lastChild
const parent = document.getElementById("demo").parentNode;

DOM Node Lists

Node lists are collections of nodes retrieved using methods like querySelectorAll().

let items = document.querySelectorAll("li");
items.forEach(item => item.style.color = "blue");

JavaScript Web APIs

Web Storage API

Used for storing data locally in the browser.

localStorage.setItem("name", "Aryan");
let name = localStorage.getItem("name");

Web Fetch API

Used to fetch resources over the network.

fetch("https://jsonplaceholder.typicode.com/todos/1")
  .then(response => response.json())
  .then(json => console.log(json));

Web Geolocation API

Used to get the geographical position of a user.

navigator.geolocation.getCurrentPosition(position => {
  alert("Latitude: " + position.coords.latitude);
});

JavaScript AJAX

AJAX is used to exchange data with a server and update parts of a webpage without reloading.

AJAX Request

const xhttp = new XMLHttpRequest();
xhttp.onload = function() {
  document.getElementById("ajaxDemo").innerHTML = this.responseText;
}
xhttp.open("GET", "https://jsonplaceholder.typicode.com/posts/1");
xhttp.send();
Data will be loaded here

JavaScript JSON

Definition

JSON (JavaScript Object Notation) is a format for storing and transporting data.

JSON Parse

const jsonStr = '{"name":"Aryan","age":20}';
const obj = JSON.parse(jsonStr);

JSON Stringify

const obj = { name: "Aryan", age: 20 };
const jsonStr = JSON.stringify(obj);

JSON vs XML

  • JSON is lighter and easier to read than XML.
  • JSON uses key-value pairs.
  • XML uses nested tags and is more verbose.
