📘 About LearnX

LearnX is a personal project developed as part of a learning journey in programming and web development. It’s designed to be a simple, user-friendly educational platform built entirely using modern web technologies.

🌟 Topics Covered:

  • 💻 C Programming: Syntax, control flow, arrays, and functions.
  • 🐍 Python: Data types, functions, and logic.
  • ☕ Java: Object-oriented programming basics.
  • 🌐 HTML & CSS: Web structure and styling.
  • 🧠 JavaScript: Interactivity and DOM manipulation.
  • 🗃️ SQL: Database queries and management.
  • 📊 Excel: Data analysis and formulas.

The goal is to make learning clear, practical, and self-paced. LearnX acts as a digital notebook and a space for applying knowledge to real-world concepts.
